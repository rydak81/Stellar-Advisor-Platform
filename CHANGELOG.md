# Changelog

All notable changes to StellarAdvisor will be documented in this file.

## [1.0.0] - 2026-01-17

### Added
- Initial release of StellarAdvisor platform
- Landing page with hero section and value proposition
- FBA Deal Calculator tool
  - Profit per unit calculation
  - Profit margin percentage
  - Monthly profit projection
  - Full cost breakdown
- Agency Match Quiz
  - 5-question assessment
  - Progress bar
  - Match results with top 3 agencies
  - Lead score display
- Lead capture modal
  - Name, email, company, phone fields
  - Form validation
- Partner section for agencies
  - Three benefit cards
  - Partnership CTA
- Mobile responsive design
- Navigation with smooth scrolling
- Tool switching tabs

### Technical
- Single HTML file architecture
- No build process required
- Google Fonts (DM Sans, Space Grotesk)
- CSS custom properties for theming
- Vanilla JavaScript (no dependencies)

## Version Format

- MINOR: New features (backwards compatible)
- PATCH: Bug fixes
